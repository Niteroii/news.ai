# Social Share
This module provides a PHP Package that allows users to share content on social networks.
Currently supported networks are:
- LinkedIn
- Github
- Twitter
- Medium
- Dev.to
- Tumblr

## Installation

### Composer
```bash
composer require cornatul/social
```
